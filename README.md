# THOI

[See Live](https://idenhan.github.io/top-100-of-itunes/)

This website shows the top 100 albums on iTunes at any given time. I have used [React](https://reactjs.org) to build it.

## Guide

To Set up:

```text
yarn
npm install
```

To develop use:

```text
yarn start
npm start
```

And It runs on

```text
localhost:3000
```

For production build use:

```text
yarn build
npm build
```
